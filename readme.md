1 概述
======

蓝牙 BLE (Bluetooth)
技术是一种无线数据和语音通信开放的全球规范，是一种高普及率、低成本的近距离无线连接协议方案。BLE
单点通用方案是涂鸦 MCU 对接的主流方案之一，其通信原理图如下。MCU 只需要对接涂鸦
BLE 通用对接协议，即可实现联网，涂鸦提供完整的模组、App 及云端服务。本文为涂鸦
BLE 单点通用方案 MCU 对接开发的过程概述。
